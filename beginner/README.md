# 🟢 Beginner Track — Task Management API

Welcome to the Beginner Track of the Task Management Intern Program.

In this track, you will build a RESTful Task Management API from scratch
using Java and Spring Boot. By the end of the program, you will have a
real-world backend project in your portfolio.

## What You Will Build

A backend API that allows users to:
- Register and log in securely (JWT Authentication)
- Create and manage projects
- Invite members to projects
- Create, assign, and track tasks within projects

## Tech Stack

| Technology | Purpose |
|---|---|
| Java 17/21 | Programming language |
| Spring Boot 4 | Application framework |
| Spring Security | Authentication & Authorization |
| Spring Data JPA | Database operations |
| PostgreSQL | Relational database |
| JWT | Token-based authentication |
| Lombok | Boilerplate reduction |
| Maven | Dependency management |

## Prerequisites

- Basic Java knowledge (variables, loops, classes)
- Git fundamentals (clone, commit, push)
- A computer with Java 17+ and Maven installed

## Program Structure

| Week | Topic | Key Outcome |
|---|---|---|
| Week 1 | Setup & Git Flow | Running Spring Boot project |
| Week 2 | Entities & Auth | Register / Login working |
| Week 3 | Project & Members | CRUD + Authorization |
| Week 4 | Tasks & Delivery | Full working API |
| Week 5 *(optional)* | Testing | %60+ test coverage |
| Week 6 *(optional)* | Swagger & Docker | Containerized app |

## How to Get Started

1. Fork this repository
2. Clone your fork to your local machine
```bash
   git clone https://github.com/YOUR_USERNAME/task-management-intern-program.git
```
3. Read [Week 1 Guide](./modules/week-1/README.md) and get started
4. Open a Pull Request at the end of each week
