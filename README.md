# Task Management Intern Program 🚀

A structured internship program designed to train backend developers
using Java & Spring Boot through a real-world Task Management API project.

## Tracks

### 🟢 Beginner Track
For interns with little or no backend experience.
Focuses on building a RESTful API from scratch using core Spring Boot concepts.

**Duration:** 4 weeks (up to 6 weeks optional)  
**Stack:** Java, Spring Boot, Spring Security, PostgreSQL, JWT  
[Go to Beginner Track →](./beginner/README.md)

### 🔴 Advanced Track
For interns with foundational backend knowledge.
Focuses on software architecture, event-driven design, and production-ready code.

**Duration:** 8–10 weeks  
**Stack:** Java, Spring Boot, Kafka, Redis, PostgreSQL, Hexagonal Architecture  
[Go to Advanced Track →](./advanced/README.md)

## How It Works

1. Intern forks this repository
2. Picks the appropriate track
3. Follows the weekly module guides
4. Opens a Pull Request at the end of each week
5. Receives code review and feedback
